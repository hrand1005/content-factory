## install python packages

```
pip3 install -r requirements.txt
```

## set up twitch cli:

```
brew install twitchdev/twitch/twitch-cli
twitch configure
twitch token
```

## set up empty files / directories? 

```
mkdir /compile/tmp
mkdir /db/tmp
touch clips.txt
```
